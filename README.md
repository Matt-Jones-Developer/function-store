## Built With

- Next.js
- Sanity.io
- Tailwind
- AlpineJS
- TypeScript
- GROQ query
- Love

Next.js - A server-side rendered, react-based framework front-end, with a sanity back-end. Sanity is a fully customizable, headless content management system that allows developers and content creators to collaborate and manage structured content in real-time. Styled with Tailwind.
